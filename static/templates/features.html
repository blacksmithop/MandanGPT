Bot Features

Discover all the powerful capabilities of MandanGPT and how it can enhance your Discord server experience.

AI & Machine Learning

Natural Conversations

Engage in natural, context-aware conversations with an advanced AI that understands nuance and provides helpful, accurate responses.

Use /ask to start a conversation

Custom Knowledge Base

Upload your own data to create a personalized knowledge base. The bot learns from your documents, websites, and text to provide tailored responses.

Use /ingest to add custom data

Contextual Memory

The bot remembers previous conversations and maintains context, allowing for more natural and coherent multi-turn dialogues.

Conversations maintain context for up to 10 messages

Server Management

Moderation Tools

Comprehensive moderation commands to manage your server, including ban, kick, mute, and message purging with detailed logs.

Requires moderator permissions

Custom Welcome Messages

Create personalized welcome messages for new members with customizable templates, role assignments, and welcome channels.

Configure with /config welcome

Role Management

Automate role assignments, create self-assign role menus, and set up role hierarchies to streamline server organization.

Use /roles commands to manage

Utility Features

Scheduling & Reminders

Schedule events, set reminders, and create recurring notifications for your server members with timezone support.

Use /remind or /schedule commands

Polls & Voting

Create interactive polls with multiple options, reaction voting, and real-time results tracking for community decisions.

Use /poll to create voting sessions

Auto-Moderation

Intelligent content filtering with customizable rules to automatically detect and handle spam, inappropriate content, and raids.

Configure with /automod commands

Entertainment & Fun

Games & Trivia

Enjoy a variety of text-based games, trivia challenges, and interactive activities to keep your server members engaged.

Try /trivia or /game commands

Memes & Media

Generate memes, fetch random images, and access a variety of media commands to share fun content in your server.

Use /meme or /image commands

Creative Generation

Generate stories, poems, jokes, and creative content on demand with AI-powered text generation capabilities.

Try /generate or /create commands
